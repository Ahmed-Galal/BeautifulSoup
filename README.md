# BeautifulSoup
web scarping using BeautifulSoup 
* clone the project :
```sh
    git clone https://github.com/Ahmed-Galal/BeautifulSoup.git
```
* open the direcotry :
```sh
    cd BeautifulSoup
```
* install the requirements :
```sh
    pip install -r requirements.txt
```
* run the scarping:
```sh
    python html_parse.py
```
* to learn more read the [docs]
